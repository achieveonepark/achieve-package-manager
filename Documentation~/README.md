# Achieve Package Manager

Unity용 UI Toolkit 기반 패키지 매니저. **OpenUPM 패키지**, **큐레이션된 GitHub 유틸리티**, **임의의 Git URL**을 클릭 한 번으로 `Packages/manifest.json`에 주입합니다.

## 한눈에 보기

* **Installed** — 매니페스트의 모든 패키지와 소스(Registry / Git / Local / Built-in)를 한 화면에. 미해결 항목도 표시되어 깨진 매니페스트를 도구에서 복구할 수 있어요.
* **OpenUPM** — OpenUPM 레지스트리 전체를 검색·필터링 후 1클릭 설치. 스코프 자동 등록.
* **Featured** — UniTask, NuGetForUnity, VContainer, DOTween 등 자주 쓰는 Git 패키지 카드 1클릭 설치.
* **Custom Git URL** — 임의의 Git URL 입력 후 설치. 이름은 자동 추정, 수동 오버라이드 가능.
