**Unity 패키지 관리, 깔끔하게.**
UI Toolkit · OpenUPM · 큐레이션 Git 패키지 · Custom Git URL을 한 창에서 관리하세요.

## 특징

- **Installed** — 매니페스트의 모든 패키지를 한 화면에 보여주고 소스(Registry / Git / Local / Built-in)를 자동 감지합니다. 미해결 항목까지 표시되어 깨진 매니페스트를 도구에서 바로 복구할 수 있어요.
- **OpenUPM** — OpenUPM 레지스트리 패키지를 검색·설치합니다.
- **Curated Git** — 큐레이션된 Git 패키지 목록을 제공합니다.
- **Custom Git URL** — 임의의 Git URL 패키지를 추가합니다.
