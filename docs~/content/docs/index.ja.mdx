**Unity のパッケージ管理を、すっきりと。**
UI Toolkit · OpenUPM · キュレーションされた Git パッケージ · Custom Git URL を 1 つのウィンドウで管理します。

## 特徴

- **Installed** — マニフェストのすべてのパッケージを 1 画面で表示し、ソース（Registry / Git / Local / Built-in）を自動検出します。未解決の項目まで表示され、壊れたマニフェストをツール内で直接修復できます。
- **OpenUPM** — OpenUPM レジストリのパッケージを検索・インストールします。
- **Curated Git** — キュレーションされた Git パッケージの一覧を提供します。
- **Custom Git URL** — 任意の Git URL パッケージを追加します。
