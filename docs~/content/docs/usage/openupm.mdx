---
title: "OpenUPM 탭"
---

[OpenUPM](https://openupm.com/) 레지스트리 전체를 탐색하고, 클릭 한 번으로 설치합니다.

## 데이터 출처

```
GET https://package.openupm.com/-/all
```

OpenUPM은 npm 호환 레지스트리(Verdaccio 기반)예요. 위 엔드포인트는 OpenUPM이 인덱싱하는 모든 패키지를 한 번에 반환합니다.

> 응답이 수 MB로 큰 편이지만, 결과는 메모리에 캐싱되므로 **세션당 1회**만 다운로드됩니다. 강제 갱신은 툴바의 `↻ Refresh`로만 발생해요.

## 검색

검색 필드는 다음 세 가지에 대해 부분 일치(대소문자 무시)로 필터링합니다.

* 패키지 ID (예: `com.cysharp.unitask`)
* Display Name
* Description

리스트는 가독성을 위해 **최대 500개까지** 표시합니다. 더 좁히려면 검색어를 추가하세요.

## 설치 동작

`Install` 버튼을 누르면:

1. `package.openupm.com` 스코프 레지스트리가 없으면 추가, 있으면 병합
2. 패키지 스코프를 자동 등록  
   예: `com.cysharp.unitask` → 스코프 `com.cysharp`
3. `dependencies`에 패키지 이름 + 최신 버전(`dist-tags.latest`) 기록
4. `Client.Resolve()` 호출

결과 메시지가 우측 하단 노티피케이션으로 표시됩니다.

## 잘못된 이름 필터

OpenUPM `/-/all`이 npm 포맷이라 `@scope/name` 같은 **UPM 규칙에 맞지 않는 이름**이 섞여 있을 수 있어요. 도구는 이를 자동으로 필터링해서 목록에서 제외합니다. (이 검증이 없다면 잘못 설치 시 매니페스트가 깨져 모든 resolve가 막힙니다.)

## 이미 설치된 패키지

설치된 패키지에는 `Installed` 배지가 표시되고, 버튼 라벨이 `Reinstall`로 바뀝니다.
