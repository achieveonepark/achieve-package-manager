# Achieve Package Manager

**Clean Unity package management.**
Manage UI Toolkit · OpenUPM · curated Git packages · custom Git URLs from a single window.

## Features

- **Installed** — Shows every package in your manifest on one screen and auto-detects its source (Registry / Git / Local / Built-in). Unresolved entries are surfaced so you can fix a broken manifest right from the tool.
- **OpenUPM** — Search and install packages from the OpenUPM registry.
- **Curated Git** — Browse a curated list of Git packages.
- **Custom Git URL** — Add any package by Git URL.
