**让 Unity 的包管理更清爽。**
在一个窗口中管理 UI Toolkit · OpenUPM · 精选 Git 包 · Custom Git URL。

## 特性

- **Installed** — 在一个界面显示清单中的所有包，并自动检测来源（Registry / Git / Local / Built-in）。连未解析的项也会显示，可在工具内直接修复损坏的清单。
- **OpenUPM** — 搜索并安装 OpenUPM 注册表的包。
- **Curated Git** — 提供精选的 Git 包列表。
- **Custom Git URL** — 添加任意 Git URL 的包。
